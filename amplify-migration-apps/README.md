# Amplify Migration Apps

Example Amplify Gen1 apps to test the Gen2 migration.

## Disclaimer

- Not all apps support migration end-to-end.
- These apps do not exhibit (nor intend to) amplify or AWS best practices. 
They were designed solely for the purpose of testing migration scenarios and cover as 
many features as possible.
